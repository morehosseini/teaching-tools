How to add a new tool to this page
1
Create a subfolder
Add a new directory (e.g. concrete-mix/) containing an index.html file.
2
Self-contained HTML
Each tool is a single standalone index.html using CDN dependencies — no build step required.
3
Add a card here
Copy a card block in this file, update the href, title, description, and tags.
4
Push to GitHub
Commit and push. GitHub Pages automatically deploys within ~60 seconds.
